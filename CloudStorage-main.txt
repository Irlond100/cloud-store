# Дипломный проект "Облачное хранилище"

Разработанное приложение представляет собой REST API приложение для хранения файлов в облаке.

# Краткая характеристика приложения

- Java 17;
- Spring Boot + Spring Security;
- PostgreSQL
- Сборщик пакетов Gradle;
- Спецификация;
- Docker;  
- Тестирование;

# Сервис аутентификации
- Аутентификационные данные пользователя хранятся в БД Postgres 
- Для аутентификации пользователя, сервис аутентификации получает пару логин/пароль и возвращает клиенту JWT токен,  
который в дальнейшем прикрепляется в заголовоки клиентских запросов, хранится на стороне клиента*
 
# JWT токен
- Выдается при аутентификации через пару логин/парооль
_ соответствующие настройки можно выбрать в `application.yml`

# Тестирование
После поле запуска FROND приложения по инструкции проект перес обирается.
Перед запуском нашего приложения необходимо остановить запущенные контейнеры Docker (команда docker-compose
stop), после чего осуществляется развёртывание нашего приложения (команда docker
-compose up --build -d).

Проверка ответа от Front

Вводим в браузере адрес, на котором должен быть развёрнут Front:

http://localhost:8081/login

Если страница открылась корректно. FROND готов для тестирования
нашего приложения.

Проверка ответа от Back. Используем login: user  и пароль: test
